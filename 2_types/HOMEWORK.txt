Валидация идентификатора python
Напишите скрипт, анализирующий пользовательский ввод (input()) на возможность использования как идентификатора python.


Если идентификатор невалиден, скрипт должен указать причину, почему:
>>> class
NOK: ключевое слово

>>> 99problems
NOK: начинается с цифры


Если идентификатор валиден, скрипт должен указать дополнительную информацию об идентификаторе:
>>> __hi
OK: так называют приватные идентификаторы

>>> MyVar
OK: так называют классы

>>> MY_VAR
OK: так называют константы

Чем больше случаев скрипт будет обрабатывать тем лучше.
