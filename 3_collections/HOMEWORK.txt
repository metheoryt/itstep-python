Вы являетесь разработчиком крупной IT компании.
Некоторое время назад было объявлено о переезде в новое здание и о том, что будет поставлено новое оборудование.

Администраторы обратились к вам с просьбой используя некоторый источник данных (текстовый файл),
исходя из знания ID, имени, фамилии, отчества, а также отдела сформировать для каждого служащего
новые логины и пароли для входа в систему.

Данные, которые предоставили вам администраторы выглядят следующим образом (пример):
1082:Anatoliy:Ivanovich:Kapec:Salary
1083:Vasiliy:Antonovich:Shmatko:HR
...


Ваша задача - обработать данные подобного рода и вывести подобную таблицу:
Full name       Id      Position    Login       Password
A. I. Kapec     1082    Salary      aikapec     1akd8ns2


Напишите такой код, который позволит создать уникальные пароли и логины для пользователей.

P.S. Данные, которые предоставлены администраторами могут быть любые,
в качестве примера создайте по приведённому выше шаблону не менее 7 записей.
