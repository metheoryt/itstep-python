Мощность языка Python в его возможном повсеместном применении.
Представьте, что требуется написать бота, который анализирует все ссылки на любой странице и выводит их пользователю.
Ваш заказчик хочет себе такого бота  для того, чтобы понять,
действительно ли рекламирующий его продукцию сайт выкладывает все ссылки по договору?

Напишите программу, которая позволяет получать текст любой html страницы,
и считать в нём появление всех ссылок с выводом отчета для пользователя.

Скрипт должен ожидать первым аргументом URL на страницу в интернете,
и возвращать количество найденных на этой странице ссылок, сгруппированых по домену.

Пример вывода:
19 www.google.com
6  itstep.org
2  metrika.yandex.ru

* Чтобы выполнить запрос в интернет, воспользуйтесь модулем requests
В консоли:
> pip install requests

В коде:
> import requests
> response = requests.get('https://example.com')
> html_text = response.text
